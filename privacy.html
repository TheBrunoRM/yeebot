Privacy Policy

Data collection

YeeBot collects data that is provided from users by commands like bio or language, and by server owners by commands like settings or roleprefix.

This data is only used for the bot to work properly for each server and user.

If server snipes are enabled in a server's settings, the bot will collect information about edited and deleted messages, and store it for 30 days after the messages have been edited or deleted for users to use the snipe command to see said information.

You can request the deletion of your or your server's data by sending a direct message to this Discord account: BrunoRM#7316.
